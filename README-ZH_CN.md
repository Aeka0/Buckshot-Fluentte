# Buckshot Fluentte

一个以 WinUI 3 Fluent Design 风格呈现的现代"微软原生风格"游戏界面。

## 环境要求

- **Visual Studio 2022** (Community 或以上) 并安装 **Windows 应用程序开发** 工作负载
- **.NET 9.0 SDK** 或更高版本
- **Windows 10 SDK** (10.0.22621.0)

## 玩法

霰弹枪内随机填充实弹和空弹。被实弹射中会损失一点生命值。
向自己射击且为空弹，则可以赢取额外一回合行动。

### 道具

放大镜：查看当前膛内弹药
手锯：当前弹药伤害翻倍
香烟：回复一点生命值
啤酒：不开火的情况下退出一枚弹药
手铐：使对手暂停一回合行动
逆转器：将当前膛内弹药的属性逆转
肾上腺素：从对方的物品池里拿取物品
一次性手机：得知非目前膛内的随机一枚弹药
过期药物：50%概率回复两点生命值，或者失去一点生命值
